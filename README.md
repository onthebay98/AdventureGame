### Adventure game built in Haskell!

"stack build" to build

"stack exec functional-adventure-exe" to run

Goal: Find the Jarate, trade it for the Golden Frying Pan, and bring it to TC Hydro to win the game!

Commands: 
  | Inventory
  | Look
  | Drop "Item Name"
  | Take "Item Name"
  | Move "Direction"
  | Exit
  | Throw "Item Name"
